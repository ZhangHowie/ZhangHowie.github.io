我的mac鼠须管配置（小鹤双拼）

发表于 2017-09-16 | | 阅读次数:

前言

最近爱上了使用双拼，原因是少数派一直不停向我推荐双拼🙃

和我们一起学双拼，码字再快一点
做少数派中的少数派：双拼输入快速入门

我忍不住了，不过一个星期过去了，打字速度，准确度都有很大提升！

这里我向大家推荐的是：mac上强大可自定义的输入法—鼠须管

我选择的双拼是小鹤双拼，原因嘛，当然是口碑比较好，
缺点是不像智能ABC和自然码那样系统性支持得那么好。由于本人使用的是mac，支持的输入法也就搜狗输入法，但是真的好丑。

之前体验过一段时间过落格输入法，但因为和小鹤闹崩的缘故，也就抛弃了。
而且网上好多鼠须管教程好像也过期了。
这里就写个教程来，层次比较浅，大牛勿喷😂
本人也在学习研究中🤗

正文

安装

好了，废话不说了，直接上教程！

首先，我们打开鼠须管的官网

这里个人习惯使用homebrew安装，就一行命令搞定。当然你也可以使用其他方式安装。🤒

如果不知道homebrew这个神器呢，移步这里，装完你会爱上它的，记得回来感谢我！
https://brew.sh/index_zh-cn.html
其实也就一行命令：

然后你就可以使用brew命令去安装软件了。

正式安装：

brew cask install squirrel

这里为了给大家演示，还重装了一次😕

装完以后，点击
Control + ` 可以切换输入方式

` 键就是数字1左边那个键

现在打开用户设定：

会进入到设置的配置文件夹

然后找到 default.yaml 文件，打开找到 schema_list:

这里就是输入配置的地方，由于鼠须管默认是没有双拼的，我们需要自己前往下载：
https://github.com/rime/rime-double-pinyin
支持 智能 ABC， 自然碼， 小鶴 等等
选择自己需要的方案，下载到配置文件夹/Users/YourUserName/Library/Rime
我这里下载的是小鹤双拼，也就是 double_pinyin_flypy.schema.yaml

接下来要做的就是，在default.yaml 指定小鹤双拼。
- schema: double_pinyin_flypy
我这里直接注释掉了不用的方案。

好的，现在已经在配置文件里面设置好了，只需要按下
Control + Alt + ` 重新部署一下

Control + ` 切换输入方式，选择小鹤双拼，就可以享受你的双拼输入了！

美化

配置文件夹里面，squirrel.yaml 文件是配置鼠须管配色主题之类的文件。

我们需要去复制一个squirrel.yaml 文件去备份（因为如果我们等下重新部署以后文件就会被修改）

重命名为：squirrel.custom.yaml，并打开它。

默认输入法中内置了很多主题，我们可以直接使用。
比如：aqua，azure，google等等

直接在 color_scheme: 后面加入就可以使用

那如果我们想加入自己自定义的配色怎么办？

那就是仿照给的主题，添加自定义的配色。（废话！）

给出我采用的配色方案，也是从别人那里整来的

然后将修改好的 squirrel.custom.yaml ，全部复制到squirrel.yaml文件中

部署一下，Bingo！

如果不满意这种配色，你自己去改呀，哈哈哈！

常用快捷键：

⌃ + ` （Control + `） 选择输入方案
⌥ + ⌃ + `（Control + Option + `） 重新部署
